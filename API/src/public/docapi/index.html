📌 Informações Importantes

🔐 Autenticação

Todos os endpoints requerem autenticação via Bearer Token, exeto os endpoints de login. Inclua no header: Authorization: Bearer {token}

📦 Formato dos Dados

Content-Type: application/json
Respostas sempre em JSON estruturado

⚠️ Códigos de Status

200: Sucesso
201: Criado
400: Erro de requisição
401: Não autorizado
404: Não encontrado
